Document Analyzer


Basic text processing app for my Software Dev class in Fall 2023.
Developed in 1 week using Java and IntelliJ.
The program reads text files and counts each occurance of every word and bigram. Then, it displays the most common entries of each.

Project Analysis


This project served as a culmination of my knowledge of Java knowledge so far. I took multiple Java classes at Gateway, so the intro to programming class at MSOE really wasn't too difficult for me. However, this project was definitely the most challenging thing we did this semester. It involved reading files, basic data structures, error handling, and basic data procesisng.

I collaborated with one of my classmates to complete this project. We were able to bounce ideas off eachother, and completed the project much faster than if we had attempted it alone. This showed me the effectiveness of coding in pairs, a strategy which I will likely employ in the future. Although software development is often a solitary discipline, working with others is always fun and a good way to improve your workflow.

Overall, I feel that this project was a good review for my java skills, which I am now very confident with. I'm extremely excited for next semester, when I'm taking data structures and algorithms. I'm confident that I will do well in this class.
