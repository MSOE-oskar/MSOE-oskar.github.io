5 Nightmares at MSOE


Horror video game created in the Unity Game Engine with C#.
Developed in one month with no prior Unity knowledge for the MSOE Game Dev Club InkJam 2023.
Ended up placing 3rd overall among 12 submissions.

Project Analysis


This project was my first time coding in a somewhat competitive setting, with a time limit against others. One thing I am proud of myself for in this project was my time management. For my school assignments I tend to procrastinate, so I was worried that I would end up waiting until the last minute to finish. However, I was able to effectively balance my time, and create plan of what to complete each week.

At the end of the competition, I had a complete game that I was rather proud of, as it was really the first major solo project I had done. One skill I used was C#, which I had taken classes in, but never really applied outside of class. I also learned the unity engine completely from scratch, so I am now confident that in the future I'll be able to make better games.

One thing I think could be improved in the future is my documentation. Because the project was small enough, I was able to get away with not commenting my code nor creating a github repository. But I realize how important it is to do these things, so in the future I plan to create github repositories and comment all my code, no matter how small the project.

Overall, I really enjoyed creating 5 Nightmares at MSOE. Video games and game dev have been a passion of mine ever since grade school, and this project feels like a big step in my personal journey.
